# 图片与位置工具

一个可以在手机中作为应用安装使用的 PWA 工具，包含：

- **获取相册所有图片**：选择相册中的图片并查看（需用户授权选择）
- **监听现在所在位置**：实时获取并显示当前 GPS 位置

## 如何用手机安装为应用

### 方法一：本地电脑开服务器，手机访问（推荐）

1. 在项目文件夹打开命令行，执行：
   ```bash
   npx serve .
   ```
2. 电脑会显示一个地址，如 `http://192.168.x.x:3000`
3. 用手机连接与电脑相同的 Wi‑Fi，在手机浏览器中打开该地址
4. 在浏览器菜单中选择「添加到主屏幕」或「安装应用」

### 方法二：部署到网上后访问

将整个项目上传到支持 HTTPS 的服务器（如 GitHub Pages、Vercel、Netlify 等），在手机浏览器中访问你的网址，然后选择「添加到主屏幕」。

## 使用说明

- **获取相册图片**：点击按钮后，从相册选择多张图片，会在页面中显示
- **监听位置**：点击后授权位置权限，即可实时显示当前位置；再次点击可停止监听

## 注意事项

- 相册图片需在手机浏览器中手动选择（出于隐私，浏览器不会自动获取全部相册）
- 位置功能需要手机开启定位，并授予浏览器定位权限
